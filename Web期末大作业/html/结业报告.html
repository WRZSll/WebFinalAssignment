开发过程

开发的大部分时间是在网上学习前端的相关知识，编写JavaScript、HTML、CSS、TypeScript的笔记以及收藏学习自己觉得好看的前端界面，并应用到自己的前端项目， 期间遇到一些问题，在网上寻找方法解决。

遇到的问题


内边距问题：行内元素margin和padding，无法改变元素垂直位置

选择器优先级问题：未考虑选择器的优先级，导致页面呈现的效果与预期大相径庭

合并现象：垂直排列的兄弟元素，上下margin会合并

塌陷问题：父子级的标签，子级的添加上外边距导致父级一起向下移动

解决

给行内元素添加line-height可以改变垂直位置

选择器优先级的样式生效：通配符选择器 < 标签选择器 < 类选择器 < id选择器 < 行内选择器 < !important

提前计算好垂直排列的兄弟元素的上下间距

取消子级margin，父级padding，父级设置overflow:hidden，父级设置border-top

未解决的问题
